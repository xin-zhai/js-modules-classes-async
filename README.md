# js-modules-classes-and-async-homework

js modules classes and async homework

请使用 `npm run start` 命令启动项目并查看题目

## Install dependencies

```
yarn install
```

If you use npm,

```
npm install
```

## Run Test

```
yarn test
```

If you use npm,

```
npm test
```

### with Watch Mode

```
yarn test:watch
```

### with Code Coverage

```
yarn test:coverage
```

## JS and CSS code lint in src/

```
yarn run lint
```

If you use npm,

```
npm run lint
```

## Format JS and CSS files in src/

```
yarn run format
```

If you use npm,

```
npm run format
```

## NOTE

Your committed code will be automatically format and check lint when submit
