Practice2

创建两个类：Person 与 Student, Student 继承于 Person 且满足以下要求：

  1. Person 和 Student 构造实例时需要传入姓名字符串作为实例的姓名属性name，Student还需要传入班级字符串作为实例的班级属性className。
    2. 
      const kevin = new Person('Kevin')
      kevin.name // => Kevin
      const winnie = new Student('Winnie', 'class1')
      winnie.name // => Winnie
      winnie.className // => class1
  2. Person具有 move 方法, 实例调用时输出(console.log)："XXX(name) is moving", Student还具有 study 方法,实例调用时先调用 move 方法，再输出(console.log)："XXX(name) is studying in XXX(className)"
    3. 
      kevin.move() // => Kevin is moving
      winnie.study() // => Winnie is moving & Winnie is studying in class1
  3. study 方法无论怎样被调用 this 都需要指向声明的实例
    4. 
      const winnie = new Student('Winnie', 'class1')
      const { study } = winnie
      study() // => Winnie is moving & Winnie is studying in class1
